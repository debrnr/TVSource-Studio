### 步骤2: 在TVBox中加载配置

1. **打开TVBox应用**
2. **进入"配置"或"源管理"**
3. **选择"远程配置"**
4. **输入配置URL**：
   ```
   http://192.168.0.88:8080/api/tvbox/config
   ```
5. **点击"确定"或"加载"**

---

### 步骤3: 清除TVBox缓存（非常重要！）

如果之前加载过旧配置，**必须清除缓存**：

**常见清除缓存方法**（不同版本TVBox位置可能不同）：

**方法1**: 
- 设置 → 应用管理 → TVBox → 存储 → 清除缓存

**方法2**:
- 长按TVBox图标 → 应用信息 → 存储 → 清除缓存

**方法3**:
- 设置 → 数据管理 → 清除所有数据

**方法4**（终极方案）:
- 卸载TVBox
- 重新安装TVBox
- 首次加载配置

## 🔧 常见问题排查

### Q1: TVBox提示"加载失败"或"连接超时"

**可能原因**：
1. **防火墙阻止**：192.168.0.88的8080端口未开放
2. **网络不通**：TVBox设备无法访问192.168.0.88

**解决方法**：

**检查防火墙**（在192.168.0.88机器上）：
```powershell
# Windows防火墙添加规则
New-NetFirewallRule -DisplayName "TVSource Studio" -Direction Inbound -LocalPort 8080 -Protocol TCP -Action Allow
```

**测试网络连通性**：
- 在TVBox设备上ping 192.168.0.88
- 或使用另一台设备测试访问 `http://192.168.0.88:8080`

---

### Q2: TVBox加载配置成功，但仍只显示1部电影

**原因**：TVBox缓存了旧的API响应

**解决方法**：
1. **清除TVBox缓存**（见步骤3）
2. **重启TVBox应用**
3. **重新加载配置**

---

### Q3: 如何确认API真的返回21部电影？

**在本机测试**：
```powershell
Invoke-RestMethod -Uri "http://192.168.0.88:8080/api/maccms/vod?ac=list&t=1&pg=1" | ConvertTo-Json
```

应该看到：
```json
{
  "code": 1,
  "msg": "聚合数据（1条API + 20条k1k）",
  "total": 21,
  "list": [ ... 21部影片 ... ]
}
```
