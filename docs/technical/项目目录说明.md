## 🎯 各目录详细说明

### 1. 根目录（Root）

**用途**：存放启动入口文件和全局配置

| 文件 | 说明 | 是否必需 |
|------|------|----------|
| `start.bat` | **一键启动脚本**，双击即可运行服务 | ✅ 必需 |
| `activate_env.bat` | 激活Python虚拟环境 | ✅ 必需 |
| `.gitignore` | Git忽略规则 | ✅ 必需 |
| `.gitattributes` | Git行尾符规范 | 推荐 |
| `LICENSE` | MIT许可证 | 推荐 |

**注意**：保持根目录简洁，仅保留最常用的文件。

---

### 2. config/ - 配置文件

**用途**：集中管理所有配置文件

| 文件 | 说明 | 重要性 |
|------|------|--------|
| `.env` | **环境变量配置**，包含EXTERNAL_HOST、端口等 | ⭐⭐⭐⭐⭐ |
| `.env.example` | 配置模板，首次使用时复制为`.env` | ⭐⭐⭐⭐⭐ |
| `requirements.txt` | Python依赖包列表 | ⭐⭐⭐⭐⭐ |
| `settings.py` | 应用常量配置 | ⭐⭐⭐⭐ |

**首次使用步骤**：
```powershell
# 1. 复制模板
copy config\.env.example config\.env

# 2. 编辑配置
notepad config\.env

# 3. 修改 EXTERNAL_HOST 为您的机器IP
```

---

### 3. src/ - 核心源代码

**用途**：Flask应用的核心业务逻辑

#### 主要文件

| 文件 | 说明 | 功能 |
|------|------|------|
| `app.py` | Flask应用入口 | 初始化应用、启动HTTP服务、定时任务调度 |
| `routes.py` | API路由定义 | MacCMS API代理、TVBox配置生成、缓存管理 |

#### 子目录

**tasks/** - 后台任务模块
- `source_collector.py` - 自动收集和验证影视源
- `epg_updater.py` - 定时更新EPG节目单

**utils/** - 工具函数模块
- `cache_manager.py` - SQLite缓存管理

---

### 4. tools/ - 核心工具脚本

**用途**：独立运行的维护和测试工具

| 脚本 | 说明 | 使用场景 |
|------|------|----------|
| `k1k_adapter.py` | k1k.cc爬虫适配器 | 被routes.py调用，补充影片数据 |
| `health_check_api_sources.py` | API源健康检查 | 定期测试MacCMS API源可用性 |

**使用方法**：
```powershell
# 健康检查
.\venv\Scripts\python.exe tools/health_check_api_sources.py
```

---

### 5. data/ - 运行时数据

**用途**：存储运行时生成的数据文件

## 🚀 快速开始

### 1. 首次使用

```powershell
# 1. 激活虚拟环境
.\activate_env.bat

# 2. 安装依赖
pip install -r config/requirements.txt

# 3. 配置环境变量
copy config\.env.example config\.env
notepad config\.env
# 修改 EXTERNAL_HOST 为您的机器IP

# 4. 启动服务
.\start.bat
```

### 2. 访问服务

- **Web界面**: http://192.168.0.88:8080
- **TVBox配置**: http://192.168.0.88:8080/api/tvbox/config
- **M3U直播源**: http://192.168.0.88:8080/iptv/live.m3u

## 🔧 常用维护任务

### 1. 检查API源健康状态

```powershell
.\venv\Scripts\python.exe tools/health_check_api_sources.py
```

结果保存在：`data/results/healthy_maccms_sources.json`

### 2. 清理缓存

```powershell
# 方法1：通过API
Invoke-RestMethod -Uri "http://127.0.0.1:8080/api/cache/clear" -Method POST

# 方法2：删除数据库文件
Remove-Item data/maccms_cache.db
```

### 3. 查看日志

```powershell
Get-Content logs/app.log -Tail 50
```

### 4. 更新EPG

```powershell
# 手动触发EPG更新
Invoke-RestMethod -Uri "http://127.0.0.1:8080/api/epg/update" -Method POST
```

## ⚠️ 重要提示

### 1. EXTERNAL_HOST配置

**必须正确配置** `config/.env` 文件中的 `EXTERNAL_HOST`：

```ini
# 如果Flask服务运行在 192.168.0.88
EXTERNAL_HOST=http://192.168.0.88:8080

# 如果部署到群晖NAS（假设IP为192.168.0.100）
EXTERNAL_HOST=http://192.168.0.100:8080
```

**错误配置会导致TVBox无法加载数据！**

### 2. 防火墙设置

确保Windows防火墙允许8080端口入站：

```powershell
New-NetFirewallRule -DisplayName "TVSource Studio" -Direction Inbound -LocalPort 8080 -Protocol TCP -Action Allow
```

### 3. 虚拟环境

**严禁使用系统全局Python环境**，必须使用项目根目录的 `venv/`。

## 📊 当前系统架构

### 数据流

```
TVBox客户端
    ↓
/api/tvbox/config (返回JSON配置)
    ↓
🎬 TVSOURCE聚合站点
    ↓
/api/maccms/vod (MacCMS API代理)
    ├─ 优先: MacCMS API多源聚合
    │   └─ subocaiji.com, wujinzy.com, etc.
    ├─ 补充: k1k.cc网页爬虫
    │   └─ 每分类额外20条数据
    └─ 兜底: 模拟数据
         └─ 所有源失败时使用
```

### 数据构成

| 分类 | MacCMS API | k1k.cc | 总计 |
|------|-----------|--------|------|
| 电影 | 1条 | 20条 | **21部** |
| 连续剧 | 1条 | 20条 | **21部** |
| 综艺 | 13条 | 20条 | **33部** |
| 动漫 | 20条 | 20条 | **40部** |
