## 🚀 快速开始

### 1. 首次使用

```powershell
# 激活虚拟环境
.\activate_env.bat

# 安装依赖
pip install -r config/requirements.txt

# 修改.env配置
# 将 EXTERNAL_HOST 改为您的机器IP
```

### 2. 启动服务

```powershell
# 方法1：使用启动脚本（推荐）
.\start.bat

# 方法2：手动启动
.\venv\Scripts\python.exe src/app.py
```

### 3. 访问服务

- **Web界面**: http://192.168.0.88:8080
- **TVBox配置**: http://192.168.0.88:8080/api/tvbox/config
- **M3U直播源**: http://192.168.0.88:8080/iptv/live.m3u

## 📊 当前系统架构

### 数据流

```
TVBox客户端
    ↓
/api/tvbox/config (返回JSON配置)
    ↓
🎬 TVSOURCE聚合站点
    ↓
/api/maccms/vod (MacCMS API代理)
    ├─ 优先: MacCMS API多源聚合
    │   └─ subocaiji.com, wujinzy.com, etc.
    ├─ 补充: k1k.cc网页爬虫
    │   └─ 每分类额外20条数据
    └─ 兜底: 模拟数据
         └─ 所有源失败时使用
```

### 数据构成

| 分类 | MacCMS API | k1k.cc | 总计 |
|------|-----------|--------|------|
| 电影 | 1条 | 20条 | **21部** |
| 连续剧 | 1条 | 20条 | **21部** |
| 综艺 | 13条 | 20条 | **33部** |
| 动漫 | 20条 | 20条 | **40部** |

## 🔧 常用维护任务

### 1. 检查API源健康状态

```powershell
.\venv\Scripts\python.exe tools/health_check_api_sources.py
```

### 2. 清理缓存

```powershell
# 方法1：通过API
Invoke-RestMethod -Uri "http://127.0.0.1:8080/api/cache/clear" -Method POST

```

```powershell
Get-Content logs/app.log -Tail 50
```

```powershell

```

## ⚠️ 重要提示

### 1. EXTERNAL_HOST配置

**必须正确配置** `.env` 文件中的 `EXTERNAL_HOST`：

```ini
# 如果Flask服务运行在 192.168.0.88
EXTERNAL_HOST=http://192.168.0.88:8080

# 如果部署到群晖NAS（假设IP为192.168.0.100）
EXTERNAL_HOST=http://192.168.0.100:8080
```

**错误配置会导致TVBox无法加载数据！**

### 2. 防火墙设置

确保Windows防火墙允许8080端口入站：

```powershell
New-NetFirewallRule -DisplayName "TVSource Studio" -Direction Inbound -LocalPort 8080 -Protocol TCP -Action Allow
```

### 3. 虚拟环境

**严禁使用系统全局Python环境**，必须使用项目根目录的 `venv/`。
